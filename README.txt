EN üìå Installing Return Launchpad

1. Drag the Return Launchpad.app to the Applications folder.
2. On first launch, the system may show a warning "Cannot open the program".
   
   Choose one of the methods:
   
   Method 1 (Recommended):
   - Right-click on the application in the Applications folder
   - Select "Open" from the context menu
   - Confirm opening in the security dialog
   
   Method 2 (Via Terminal):
   - Open Terminal and run the command:
     xattr -cr "/Applications/Return Launchpad.app"
   - Then launch the application normally

3. After the first successful launch, the application will work as usual.
